# Contributing to Somnigraph Tracker

Thank you for your interest in contributing to the Somnigraph project! This tracker is designed to collect, discuss, and organize ideas from the community.

## How to Contribute

### Submitting Ideas

1. **Search First**: Before submitting a new idea, search existing issues to avoid duplicates
2. **Choose the Right Template**: Select the appropriate issue template for your submission:
   - üí° **Idea Submission** - For new concepts and proposals
   - üöÄ **Feature Request** - For specific feature requests
   - üêõ **Bug Report** - For reporting issues

3. **Be Clear and Detailed**: Provide as much information as possible
   - Clear title that summarizes the idea
   - Detailed description
   - Use cases and examples
   - Any relevant mockups, diagrams, or references

4. **Be Respectful**: Treat others with respect and be open to feedback

### Participating in Discussions

- **Add Value**: Contribute meaningful insights to discussions
- **Stay On Topic**: Keep comments relevant to the issue
- **Use Reactions**: Use üëç to show support without adding noise
- **Be Constructive**: Offer solutions and alternatives when critiquing ideas
- **Ask Questions**: If something is unclear, ask for clarification

### Issue Lifecycle

1. **New Issues**: Labeled with `needs-triage`
2. **Under Review**: Being evaluated by maintainers
3. **Accepted**: Approved for consideration/implementation
4. **In Progress**: Actively being worked on
5. **Completed**: Implemented or resolved
6. **Declined**: Not planned for implementation (with explanation)

## Labels

Issues are organized using labels:

- `idea` - New ideas and concepts
- `enhancement` - Feature requests
- `bug` - Bug reports
- `discussion` - General discussions
- `needs-triage` - Needs initial review
- `accepted` - Approved for consideration
- `in-progress` - Being worked on
- `good-first-issue` - Good for newcomers
- `help-wanted` - Looking for contributors
- `wontfix` - Will not be implemented

## Code of Conduct

This project follows a Code of Conduct. By participating, you agree to uphold this code. Please report unacceptable behavior.

## Questions?

If you have questions about contributing, feel free to open a discussion issue or reach out to the maintainers.
